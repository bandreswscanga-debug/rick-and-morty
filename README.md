# Estado Global - Provider
En esta parte del proyecto se implementó el manejo de estado global usando Provider.

Se creó la clase CharacterProvider, encargada de:

Gestionar la lista de personajes favoritos.
Permitir agregar o eliminar favoritos.
Verificar si un personaje es favorito.
Se utilizó notifyListeners() para actualizar la interfaz en tiempo real cuando hay cambios en los datos.
